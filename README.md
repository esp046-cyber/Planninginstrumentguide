# Planninginstrumentguide
comprehensive HTML mobile app for Planning Instrument Engineers working on mega refinery and oil & gas construction projects in 2026. Here's what I've included:
Key Features:
6 Main Sections: Instructions, Step Guide, Skills, Control Systems, Integration, and Automation
Mobile-Optimized: 1% margins, small fonts, responsive design
Interactive Elements: Checkboxes for tracking progress through procedures
Professional Styling: Oil & gas industry theme with blue/dark gradients
Practical Content: Real-world guidance for refinery projects
Content Highlights:
Instructions: Project planning overview, document management, coordination activities
Step-by-Step Guide: All 5 project phases from Basic Engineering through Commissioning
Skills Guide: Technical competencies, software proficiency, safety standards
Control Systems: DCS planning, SIS requirements, network infrastructure
Integration: Coordination with all engineering disciplines
Automation: APC, fieldbus technology, smart instrumentation, IOC planning
The app includes interactive checklists, warning boxes for critical safety items, and skill badges - all optimized for mobile viewing with the compact layout you requested.
